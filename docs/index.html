Was ist LH2GPX?

LH2GPX ist eine iOS-App, die Ihren eigenen Standortverlauf – zum Beispiel aus einem Google-Takeout-Export – direkt auf Ihrem iPhone importiert, übersichtlich darstellt und in gängige GPS-Formate exportiert.

Alle Verarbeitungen finden lokal auf Ihrem Gerät statt. Kein Konto, kein Cloud-Zwang, kein Tracking.

Kernfunktionen

Import Google Takeout JSON/ZIP, GPX, TCX
Übersicht Karte, Tage, Strecken, Statistiken
Einblicke Distanzen, Zeiträume, Aktivitätsauswertung
Export GPX, KML, KMZ, GeoJSON, CSV
Live-Aufzeichnung Optional, nutzerkonfiguriert, standardmäßig aus
Widget iOS-Homescreen-Widget

Datenschutzfreundlich

App Store Datenschutz: „Keine Daten erfasst"

  • Keine Kontoregistrierung erforderlich
  • Kein Tracking, keine Werbung, keine Analytics
  • Alle Daten bleiben lokal auf Ihrem Gerät
  • Optionaler Live-Upload nur an eigene nutzerkonfigurierte Adresse
  • Ausschließlich systemseitige Verschlüsselung (HTTPS/TLS via URLSession)
